# Guia-ISO27001-SGSI
Este repositorio contiene una guía práctica y estructurada sobre la norma ISO/IEC 27001 y la implantación de un Sistema de Gestión de Seguridad de la Información (SGSI) en organizaciones.

El documento resume los conceptos esenciales de la familia ISO 27000, el enfoque basado en riesgos, la definición del alcance, el liderazgo, la planificación, los controles de seguridad, la Declaración de Aplicabilidad, la auditoría interna, la revisión por la dirección y la mejora continua.

Su objetivo es servir como material de consulta para comprender cómo aplicar ISO 27001 en un entorno empresarial, facilitando una visión clara, práctica y orientada a la implantación real de un SGSI.
